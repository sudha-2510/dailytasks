HTML
It is used to create base structure in the web pages.And it is a markup laanguage.
How to create html page?
  1. For creating html page we can use mainly three words
  2. That are < html >< head > < body >
  3. Html element is a root element
  4. The combination of both opening tag and closing tag is called an element
Heading tags:
  1. h1 defines most important heading
  2. h2 heading 2
  3. h3 heading 3
  4. h4 heading 4
  5. h5 heading 5
  6. h6 defines least important heading
Formatting tags:
  1. b tag-bold text
  2. i tag-italic text
  3. strong tag-important information
  4. u tag-underline
  5. sub tag-appears the text below the normal line
  6. sup tag-appears the text above the normal line
BLock line and Inline elements:
  • A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
  • An inline element does not start on a new line. An inline element only takes up as much width as necessary.
